About FuturistAi

FuturistAi is a lightweight Python-based Flask web application that allows users to input a prompt through a web interface and receive a response generated by OpenAI's GPT model. The app is built using Python and Flask, and it integrates OpenAI's official Python SDK to interact with their API.

  • Programming Language: Python 3
  • Web Framework: Flask
  • Templating Engine: Jinja2
  • OpenAI API: Accessed through openai-python SDK
  • Frontend: HTML5 and Bootstrap 5
  • Deployment: Compatible with Render.com
  • Environment Variables: Managed with python-dotenv

The app includes a simple form on the homepage for entering prompts, a backend that calls the OpenAI API to generate responses, and a user-friendly Bootstrap interface. It demonstrates practical use of AI in web development.
